# Installation

## Clone
SSH:
```
git clone git@github.com:will-lynas/.dotfiles.git
```
HTTP:
```
git clone https://github.com/will-lynas/.dotfiles.git
```

## Install script
`./install/install.sh` \
Then either `bash` or `zsh`

## Tmux
`<prefix>I` to install plugins

## Patched font
Use Deja Vu Sans Mono Nerd Font
